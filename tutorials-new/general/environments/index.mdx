# Live and Test Environments

Gameball provides separate environments for testing and production to ensure safe development and deployment of your loyalty program.

## Test Environment

Use the test environment for:
- Development and integration
- Testing new features
- QA and staging

## Live Environment

The live environment is for:
- Production deployment
- Real customer interactions
- Actual point accrual and redemptions

## Switching Between Environments

Learn how to configure your API keys and SDK initialization for different environments.
