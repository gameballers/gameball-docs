# Entities

Learn about the fundamental data structures and entities in the Gameball platform.

## Core Entities

- **Attributes** - Customer properties and custom data points
- **Events** - Customer actions and behaviors that trigger rewards
- **Coupons** - Discount codes generated from loyalty points
- **Promotions** - Special offers and campaigns
- **Orders** - Purchase transactions for e-commerce
- **Payments** - Transactions for non-commerce businesses
