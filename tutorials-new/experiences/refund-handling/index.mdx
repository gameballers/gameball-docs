# Refund Handling

Learn how to handle order refunds and reverse point awards properly.

## Refund Types

- Full refunds
- Partial refunds
- Item-specific refunds

## Point Reversal

When processing refunds:
- Deduct awarded points
- Handle insufficient balance
- Update customer tier progress
- Cancel related coupons
