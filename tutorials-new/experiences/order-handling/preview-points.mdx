# Previewing Potential Points Before Purchase

Display potential point earnings on product pages, cart, or checkout to encourage purchases.

## Use Cases

- Product detail pages
- Shopping cart
- Checkout page
- Category pages

## Implementation

Use the Calculate Order Cashback API to preview points without actually creating an order.
