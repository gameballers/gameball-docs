# Payment Handling

Track payments and subscriptions for non-commerce businesses like SaaS, services, and subscriptions.

## Use Cases

- Subscription services
- SaaS platforms
- Service providers
- Membership programs
- Utilities

## Implementation

Use the Payment API instead of Order API for non-product-based transactions.
