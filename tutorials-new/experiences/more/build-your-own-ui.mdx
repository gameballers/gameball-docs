# Build Your Own UI

Build completely custom loyalty interfaces using Gameball APIs.

## What You Can Build

- Custom reward displays
- Branded redemption flows
- Mobile app integrations
- Custom dashboards
- Embedded widgets

## APIs to Use

- Customer Progress APIs
- Configuration APIs
- Transaction APIs
- Coupon Management APIs
