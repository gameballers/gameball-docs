# Tracking Events

Learn how to track customer events to trigger rewards, challenges, and automation campaigns.

## Overview

Events are the foundation of your loyalty program. They represent customer actions like:
- Sign ups
- Product views
- Social shares
- App opens
- Custom actions

## Implementation

Events can be tracked through:
- SDKs (Web, iOS, Android, React Native, Flutter)
- REST API
- Webhooks
