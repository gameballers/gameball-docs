## What you'll accomplish

By completing this Flutter integration, you'll be able to:

- **Enhance Customer Experience** - Provide a personalized loyalty experience through the Gameball Flutter SDK
- **Reward Customer Actions** - Automatically track and reward customer behaviors like purchases and app interactions
- **Drive Engagement** - Use cashback rewards and achievements to encourage repeat usage
- **Enable Redemptions** - Allow customers to easily use their earned points within your app
- **Grow Through Referrals** - Implement a referral system that rewards customers for bringing new users

## Prerequisites

Before starting your Flutter integration:

- Have your Gameball API credentials ready (API Key and Transaction Key)
- Flutter 3.0.0 or later
- Dart 2.17.0 or later
- iOS 13.0+ and Android 5.0+ deployment targets
- Xcode 12.0+ (for iOS development)
- Android Studio Arctic Fox+ (for Android development)
- Familiarity with Flutter and Dart development

<Note>
The Gameball Flutter SDK provides a unified API that works across both iOS and Android platforms with native performance.
</Note>

## Quick Start

If you want to get up and running quickly:

<Steps>
<Step title="Install the SDK">
  Add Gameball to your Flutter project:
  
  ```yaml
  # In your pubspec.yaml
  dependencies:
    gameball_flutter: ^4.0.0
  ```
  
  Then run:
  ```bash
  flutter pub get
  ```
</Step>

<Step title="Initialize Gameball">
  Initialize Gameball in your main.dart:
  
  ```dart
  import 'package:gameball_flutter/gameball_flutter.dart';
  
  void main() {
    runApp(MyApp());
  }
  
  class MyApp extends StatelessWidget {
    @override
    Widget build(BuildContext context) {
      // Initialize Gameball
      Gameball.configure(
        apiKey: 'YOUR_API_KEY',
        transactionKey: 'YOUR_TRANSACTION_KEY',
      );
      
      return MaterialApp(
        title: 'My App',
        home: MyHomePage(),
      );
    }
  }
  ```
</Step>

<Step title="Identify Customer">
  Identify a customer when they log in:
  
  ```dart
  Gameball.identify(
    customerId: 'customer_123',
    email: 'customer@example.com',
  );
  ```
</Step>
</Steps>
