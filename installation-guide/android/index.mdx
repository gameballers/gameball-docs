## What you'll accomplish

By completing this Android integration, you'll be able to:

- **Enhance Customer Experience** - Provide a personalized loyalty experience through the Gameball Android SDK
- **Reward Customer Actions** - Automatically track and reward customer behaviors like purchases and app interactions
- **Drive Engagement** - Use cashback rewards and achievements to encourage repeat usage
- **Enable Redemptions** - Allow customers to easily use their earned points within your app
- **Grow Through Referrals** - Implement a referral system that rewards customers for bringing new users

## Prerequisites

Before starting your Android integration:

- Have your Gameball API credentials ready (API Key and Transaction Key)
- Android Studio Arctic Fox or later
- Minimum SDK version 21 (Android 5.0)
- Target SDK version 33 or later
- Kotlin 1.7.0 or later
- Familiarity with Android development and Kotlin/Java

<Note>
The Gameball Android SDK is compatible with Android 5.0+ and supports both Kotlin and Java projects.
</Note>

## Quick Start

If you want to get up and running quickly:

<Steps>
<Step title="Install the SDK">
  Add Gameball to your project using Gradle:
  
  ```gradle
  // In your app-level build.gradle
  dependencies {
      implementation 'co.gameball:gameball-android-sdk:4.0.0'
  }
  ```
</Step>

<Step title="Initialize Gameball">
  Initialize Gameball in your Application class:
  
  ```kotlin
  import co.gameball.Gameball
  
  class MyApplication : Application() {
      override fun onCreate() {
          super.onCreate()
          Gameball.configure(
              apiKey = "YOUR_API_KEY",
              transactionKey = "YOUR_TRANSACTION_KEY"
          )
      }
  }
  ```
</Step>

<Step title="Identify Customer">
  Identify a customer when they log in:
  
  ```kotlin
  Gameball.identify(customerId = "customer_123", email = "customer@example.com")
  ```
</Step>
</Steps>
