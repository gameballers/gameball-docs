## Firebase Integration

To enable push notifications, you need to integrate Firebase Cloud Messaging (FCM) or Huawei Push Kit:

<Steps>
<Step title="Install Firebase/Huawei Dependencies">
  Add Firebase or Huawei push notification dependencies to your project
</Step>

<Step title="Configure Push Service">
  Set up FCM or Huawei Push Kit certificates and configuration
</Step>

<Step title="Get Registration Token">
  Generate FCM or Huawei token for the device
</Step>

<Step title="Register with Gameball">
  Send the token to Gameball for notification delivery
</Step>
</Steps>
