## Firebase Integration

To enable push notifications, you need to integrate Firebase Cloud Messaging (FCM):

<Steps>
<Step title="Install Firebase">
  Add Firebase SDK to your iOS project
</Step>

<Step title="Configure APNs">
  Set up Apple Push Notification service certificates
</Step>

<Step title="Get FCM Token">
  Generate FCM registration token for the device
</Step>

<Step title="Register with Gameball">
  Send the FCM token to Gameball for notification delivery
</Step>
</Steps>
