---
title: "Customer Notifications Count"
description: "Retrieve the total count of notifications for a specific customer in Gameball, providing the number of notifications matching the specified criteria."
openapi: 'GET /api/v4.0/integrations/customers/{customerId}/notifications/count'
languages: ["curl", "javascript", "python", "csharp"]
---

This API retrieves the total count of notifications for a specific customer in Gameball, identified by customerId. It provides the number of notifications matching the specified criteria without returning detailed notification records.

<Info>
**Security**: Requires **apikey** header.
</Info>
