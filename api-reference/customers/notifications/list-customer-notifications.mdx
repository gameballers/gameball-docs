## List Customer Notifications

Retrieve all notifications for the specified customer.

<Info>
Security: Provide both `apikey` and `secretkey` headers.
</Info>

<Note>
Concepts and examples adapted from the official Gameball Customers documentation: https://developer.gameball.co/rest-api/customers
</Note>

---

### Overview

Use this endpoint to build a notification center or inbox for customers.


### Parameters

Path

<ParamField path="customerId" type="string" required>
Unique identifier of the customer.
</ParamField>

Query

<ParamField query="limit" type="integer" default="50">
Maximum number of notifications to return.
</ParamField>

<ParamField query="cursor" type="string">
Pagination cursor returned from a previous call.
</ParamField>

---

### Request

<RequestExample>
```bash cURL
curl -X GET "https://api.gameball.co/api/v4.0/integrations/customers/1848877205/notifications?limit=50" \
  -H "apikey: YOUR_API_KEY" \
  -H "secretkey: YOUR_SECRET_KEY"
```
</RequestExample>

const res = await fetch(`https://api.gameball.co/api/v4.0/integrations/customers/${customerId}/notifications`, {

```python Python
import requests, os
url = f"https://api.gameball.co/api/v4.0/integrations/customers/{customer_id}/notifications"
headers = {"apikey": os.environ["API_KEY"], "secretkey": os.environ["SECRET_KEY"]}
data = requests.get(url, headers=headers).json()
```

```csharp C#
var client = new HttpClient();
client.DefaultRequestHeaders.Add("apikey", Environment.GetEnvironmentVariable("API_KEY"));
client.DefaultRequestHeaders.Add("secretkey", Environment.GetEnvironmentVariable("SECRET_KEY"));
var resp = await client.GetAsync($"https://api.gameball.co/api/v4.0/integrations/customers/{customerId}/notifications");

```

```php PHP
<?php
$url = "https://api.gameball.co/api/v4.0/integrations/customers/{$customerId}/notifications";
$ch = curl_init($url);
curl_setopt_array($ch, [
  CURLOPT_HTTPHEADER => [
    'apikey: ' . getenv('API_KEY'),
    'secretkey: ' . getenv('SECRET_KEY')
  ],
  CURLOPT_RETURNTRANSFER => true
]);
$resp = curl_exec($ch);
if ($resp === false) throw new Exception(curl_error($ch));
curl_close($ch);
$data = json_decode($resp, true);
```
