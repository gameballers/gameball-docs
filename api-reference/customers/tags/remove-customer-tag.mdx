## Remove Customer Tag

Remove a single tag from the specified customer.

<Info>
Security: Provide both `apikey` and `secretkey` headers.
</Info>

<Note>
Concepts and examples adapted from the official Gameball Customers documentation: https://developer.gameball.co/rest-api/customers
</Note>

---

### Overview

Use this endpoint to remove obsolete tags and keep segmentation accurate.


### Parameters

Path

<ParamField path="customerId" type="string" required>
Unique identifier of the customer.
</ParamField>

<ParamField path="tag" type="string" required>
The tag to be removed from the customer.
</ParamField>

---

### Request

<RequestExample>
```bash cURL
curl -X DELETE "https://api.gameball.co/api/v4.0/integrations/customers/1848877205/tags/vip" \
  -H "apikey: YOUR_API_KEY" \
  -H "secretkey: YOUR_SECRET_KEY"
```
</RequestExample>

### Response

<ResponseExample>
```json Success
{ "success": true, "message": "Tag removed" }
```
</ResponseExample>

### Code examples

<CodeGroup>
```curl cURL
curl -X DELETE "https://api.gameball.co/api/v4.0/integrations/customers/{customerId}/tags/{tag}" \
  -H "apikey: YOUR_API_KEY" \
  -H "secretkey: YOUR_SECRET_KEY"
```

```javascript Node.js
await fetch(`https://api.gameball.co/api/v4.0/integrations/customers/${customerId}/tags/${tag}`, {
  method: 'DELETE', headers: { apikey: process.env.API_KEY, secretkey: process.env.SECRET_KEY }
});
```

```python Python
import requests, os
url = f"https://api.gameball.co/api/v4.0/integrations/customers/{customer_id}/tags/{tag}"
headers = {"apikey": os.environ["API_KEY"], "secretkey": os.environ["SECRET_KEY"]}
requests.delete(url, headers=headers).raise_for_status()
```

```csharp C#
var client = new HttpClient();
client.DefaultRequestHeaders.Add("apikey", Environment.GetEnvironmentVariable("API_KEY"));
client.DefaultRequestHeaders.Add("secretkey", Environment.GetEnvironmentVariable("SECRET_KEY"));
var response = await client.DeleteAsync($"https://api.gameball.co/api/v4.0/integrations/customers/{customerId}/tags/{tag}");
response.EnsureSuccessStatusCode();
```

```php PHP
<?php
$url = "https://api.gameball.co/api/v4.0/integrations/customers/{$customerId}/tags/{$tag}";
$ch = curl_init($url);
curl_setopt_array($ch, [
  CURLOPT_CUSTOMREQUEST => 'DELETE',
  CURLOPT_HTTPHEADER => [
    'apikey: ' . getenv('API_KEY'),
    'secretkey: ' . getenv('SECRET_KEY')
  ],
  CURLOPT_RETURNTRANSFER => true
]);
$resp = curl_exec($ch);
if ($resp === false) throw new Exception(curl_error($ch));
curl_close($ch);
```
</CodeGroup>
