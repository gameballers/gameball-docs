## Attach Customer Tags

Attach one or more tags to the specified customer.

<Info>
Security: Provide both `apikey` and `secretkey` headers.
</Info>

<Note>
Concepts and examples adapted from the official Gameball Customers documentation: https://developer.gameball.co/rest-api/customers
</Note>

---

### Overview

Use this endpoint to add tags for personalization, targeting, or automation.


### Parameters

Path

<ParamField path="customerId" type="string" required>
Unique identifier of the customer.
</ParamField>

Body

<ParamField body="tags" type="array[string]" required>
List of tags to attach. Tags must already exist in your workspace configuration.
</ParamField>

---

### Request

<RequestExample>
```bash cURL
curl -X POST "https://api.gameball.co/api/v4.0/integrations/customers/1848877205/tags" \
  -H "Content-Type: application/json" \
  -H "apikey: YOUR_API_KEY" \
  -H "secretkey: YOUR_SECRET_KEY" \
  -d '{"tags":["vip","newsletter"]}'
```
</RequestExample>

### Response

<ResponseExample>
```json Success
{ "success": true, "message": "Tags attached" }
```
</ResponseExample>

### Code examples

<CodeGroup>
```javascript Node.js
await fetch(`https://api.gameball.co/api/v4.0/integrations/customers/${customerId}/tags`, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    apikey: process.env.API_KEY,
    secretkey: process.env.SECRET_KEY
```

```python Python
import requests, os
url = f"https://api.gameball.co/api/v4.0/integrations/customers/{customer_id}/tags"
headers = {"Content-Type":"application/json","apikey": os.environ["API_KEY"], "secretkey": os.environ["SECRET_KEY"]}
resp = requests.post(url, json={"tags":["vip","newsletter"]}, headers=headers)
resp.raise_for_status()
```

```csharp C#
using System.Net.Http;
using System.Text;
var client = new HttpClient();
client.DefaultRequestHeaders.Add("apikey", Environment.GetEnvironmentVariable("API_KEY"));
client.DefaultRequestHeaders.Add("secretkey", Environment.GetEnvironmentVariable("SECRET_KEY"));
var content = new StringContent("{\"tags\":[\"vip\",\"newsletter\"]}", Encoding.UTF8, "application/json");
var response = await client.PostAsync($"https://api.gameball.co/api/v4.0/integrations/customers/{customerId}/tags", content);
response.EnsureSuccessStatusCode();
```

```curl cURL
curl -X POST "https://api.gameball.co/api/v4.0/integrations/customers/{customerId}/tags" \
  -H "Content-Type: application/json" \
  -H "apikey: YOUR_API_KEY" \
  -H "secretkey: YOUR_SECRET_KEY" \
  -d '{"tags":["vip","newsletter"]}'
```

```php PHP
<?php
$url = "https://api.gameball.co/api/v4.0/integrations/customers/{$customerId}/tags";
$payload = json_encode(["tags" => ["vip", "newsletter"]]);
$ch = curl_init($url);
curl_setopt_array($ch, [
  CURLOPT_POST => true,
  CURLOPT_HTTPHEADER => [
    'Content-Type: application/json',
    'apikey: ' . getenv('API_KEY'),
    'secretkey: ' . getenv('SECRET_KEY')
  ],
  CURLOPT_POSTFIELDS => $payload,
  CURLOPT_RETURNTRANSFER => true
]);
$resp = curl_exec($ch);
if ($resp === false) throw new Exception(curl_error($ch));
curl_close($ch);
```
