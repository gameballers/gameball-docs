---
title: 'Get Customer Hash'
description: 'Generate a hash for an existing customer based on their unique customerId.'
openapi: 'GET /api/v4.0/integrations/customers/{customerId}/hash'
languages: ["curl", "javascript", "python", "csharp"]
---

The API call generates a hash for an existing customer based on their unique customerId. This hash is used to securely redeem loyalty points during transactions.

<Warning>
**Security**: Requires **apikey** and **secretkey** headers.
</Warning>
