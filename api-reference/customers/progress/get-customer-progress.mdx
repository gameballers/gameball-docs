## Get Customer Progress

Retrieve loyalty progress, including current tier, points balance, and referral stats.

<Info>
Security: Provide both `apikey` and `secretkey` headers.
</Info>

<Note>
Concepts and examples adapted from the official Gameball Customers documentation: https://developer.gameball.co/rest-api/customers
</Note>

---

### Overview

Use this endpoint to fetch a customer's program progress for building UI like balances, tiers, and referrals.


### Parameters

Path

<ParamField path="customerId" type="string" required>
Unique identifier of the customer.
</ParamField>

Query

<ParamField query="expand" type="string" default="none">
Optional comma-separated expansions. Supported: `tier`, `referrals`.
</ParamField>

---

### Request

<RequestExample>
```bash cURL
curl -X GET "https://api.gameball.co/api/v4.0/integrations/customers/1848877205/progress?expand=tier,referrals" \
  -H "apikey: YOUR_API_KEY" \
  -H "secretkey: YOUR_SECRET_KEY"
```
</RequestExample>

### Response

<ResponseExample>
```json Success
{
  "customerId": "1848877205",
  "pointsBalance": 1250,
  "tier": {
    "name": "Gold",
    "rank": 3
  },
  "referrals": {
    "total": 12,
    "successful": 7
  }
}
```
</ResponseExample>


```python Python
import requests, os

url = f"https://api.gameball.co/api/v4.0/integrations/customers/{customer_id}/progress"
headers = {"apikey": os.environ["API_KEY"], "secretkey": os.environ["SECRET_KEY"]}
resp = requests.get(url, headers=headers)
resp.raise_for_status()
data = resp.json()
```

```csharp C#

```

```php PHP
<?php
$url = "https://api.gameball.co/api/v4.0/integrations/customers/{$customerId}/progress";

  CURLOPT_HTTPHEADER => [

  ],
  CURLOPT_RETURNTRANSFER => true
]);

curl_close($ch);
$data = json_decode($resp, true);
```
