---
title: "Customer Referrals Count"
description: "Retrieve the total count of customers referred by a specified customer in Gameball, providing the number of completed and pending referrals."
openapi: 'GET /api/v4.0/integrations/customers/{customerId}/referrals/count'
languages: ["curl", "javascript", "python", "csharp"]
---

This API retrieves the total count of customers referred by a specified customer in Gameball, providing the number of completed and pending referrals.

<Warning>
**Security**: Requires **apikey** and **secretkey** headers.
</Warning>
