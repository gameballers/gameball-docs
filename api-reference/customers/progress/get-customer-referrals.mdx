---
title: "Customer Referrals"
description: "Retrieve a list of customers referred by a specified customer in Gameball, including each referral's join date and current status."
openapi: 'GET /api/v4.0/integrations/customers/{customerId}/referrals'
languages: ["curl", "javascript", "python", "csharp"]
---

This API retrieves a list of customers referred by a specified customer in Gameball, including each referral's join date and current status within the referral program.

<Warning>
**Security**: Requires **apikey** and **secretkey** headers.
</Warning>
