Fetch key reward settings such as cashback, redemption, and coupon details. The Reward Configurations API allows you to display precise information on how customers can earn and redeem points within your loyalty program.

## Available Endpoints

- **[Cashback Configurations](/api-reference/configurations/reward-configurations/cashback-configurations)** - Retrieve cashback configuration settings
- **[Redemption Configurations](/api-reference/configurations/reward-configurations/redemption-configurations)** - Retrieve point redemption configuration settings
- **[Coupon Configurations](/api-reference/configurations/reward-configurations/coupon-configurations)** - Retrieve coupon system configuration settings
- **[Update Coupon Configurations](/api-reference/configurations/reward-configurations/update-coupon-configurations)** - Update coupon system configuration settings
