Access configuration details on campaigns, tiers, and referrals. The Program Configurations API provides all necessary data for customers to view how they can progress, engage with campaigns, and benefit from referral opportunities.

## Available Endpoints

- **[Campaigns Configurations](/api-reference/configurations/program-configurations/campaigns-configurations)** - Retrieve active campaigns configuration
- **[Tiers Configurations](/api-reference/configurations/program-configurations/tiers-configurations)** - Retrieve VIP tier configurations
- **[Referrals Configurations](/api-reference/configurations/program-configurations/referrals-configurations)** - Retrieve referral program configuration
