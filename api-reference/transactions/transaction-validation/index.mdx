Add a layer of security to redemption transactions by generating one-time passwords (OTPs) for verification. The Transaction Validation API ensures that point redemptions are authorized, protecting both customers and the platform through secure OTP validation.

## Available Endpoints

- **[Generate OTP](/api-reference/transactions/transaction-validation/generate-otp)** - Generate one-time password for transaction verification
- **[Validate OTP](/api-reference/transactions/transaction-validation/validate-otp)** - Validate OTP to authorize transaction
