Control points with hold actions, letting you place, view, or release held points before redemption. Useful for reserving points until a transaction is confirmed.

## Available Endpoints

- **[Hold Points](/api-reference/transactions/hold-management/hold)** - Place points on hold before redemption
- **[Release Hold](/api-reference/transactions/hold-management/release)** - Release held points for redemption
- **[Get Hold Details](/api-reference/transactions/hold-management/get-hold)** - Retrieve details of held points
