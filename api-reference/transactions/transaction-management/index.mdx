## Transaction Management

Track customer transactions by listing transaction histories or adding manual point adjustments. The Transaction Management API records all point activities, from rewards to deductions, providing transparency and control over customer transactions.

## Available Endpoints

- **[List Transactions](/api-reference/transactions/transaction-management/list-transactions)** - Retrieve customer transaction history
- **[Add Manual Transaction](/api-reference/transactions/transaction-management/manual-transaction)** - Add manual point adjustments
