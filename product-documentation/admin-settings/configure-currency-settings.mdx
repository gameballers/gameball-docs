## Configure Gameball Currency Settings

**All Gameball customers**

---

You can access the **Currency** settings section in **Gameball Admin Dashboard** through **Settings > Admin Settings > General > Currency Section**

<Frame>
  ![Currency Settings](/images/product-docs/admin-settings/currency-settings.png)
</Frame>

Inside the Currency section you will find an option where you can change how the currency appears on Gameball's Widget to be as a symbol

<Frame>
  ![Currency Symbol](/images/product-docs/admin-settings/currency-symbol.png)
</Frame>

The chosen currency will be applied to all currency fields in the system:

- Total Rewarded Points Equivalence in **Earn Program**
- Total Redeemed Points Equivalence in **Redeem Program**
- Points value for redemption in the default redemption rule of **Earn Configurations**
- Points Equivalence inside Gameball Widget
