## Manage Your Gameball Customer Attributes

**Platforms**

Shopify (Guru), Salla (Guru), Custom Integrations (All Plans)

---

You can access the **Player Profile Attributes** settings section in **Gameball Admin Dashboard** through **Settings** >**Admin Settings > Profile Attributes**

<Frame>
  ![Customer Attributes](/images/product-docs/admin-settings/customer-attributes.png)
</Frame>

_Read more about [Customer Profile Attributes](/product-documentation/understanding-your-customers/customer-profile/customer-profile-attributes)_

On this settings page, you will be able to see Gameball's default attributes (_Cannot be Edited/Deleted_):

- Name
- First Name
- Last Name
- Email
- Gender
- Mobile Number
- Date of Birth
- Join Date
- Country
- City
- Zip Code
- Preferred Language
- UTMs
- Devices
- Total Amount Spent
- Last Order Date
- Total Orders Count
- Average Order Value
- Channel
- Gameball Creation Date

## Add/Delete custom customer profile attribute

You can add or delete a custom attribute by scrolling down to the custom profile attributes section, then Create an attribute by adding its name and a data type from the below:

- Number
- String
- Boolean
- Date

<Frame>
  ![Custom Attributes](/images/product-docs/admin-settings/custom-attributes.png)
</Frame>

<Warning>
**Note:** While using this custom attribute in the web/mobile integration, you should use the exact same name used in defining the attribute in the settings.
</Warning>
