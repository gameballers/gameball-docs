## Upload your Stores' Products and Collections through a CSV File

If you want to create loyalty events that involve specific products or collections, Gameball will need some information about them first. To provide this data, you can easily upload a CSV file containing all the details of your products or collections. Here's how to do it.

## How to Access

1. Go to **Dashboard → Settings → Admin Settings →** **Products and Collections** under **Integrations**.

<Frame>
  ![Products and Collections Menu](/images/product-docs/admin-settings/products-collections-menu.png)
</Frame>

<Frame>
  ![Products and Collections Page](/images/product-docs/admin-settings/products-collections-page.png)
</Frame>

Under **Products** or under **Collections**, click "**Click to uploard**" or drag and drop your file.

<Frame>
  ![Upload File](/images/product-docs/admin-settings/upload-file.png)
</Frame>

Here is the CSV file format for you need to follow for the **Products**:

- Column 1: Product ID
- Column 2: Product Name
- Column 3: Variant ID
- Column 4: Variant Name
- Column 5: Price
- Column 6: URL

<Frame>
  ![Products CSV Format](/images/product-docs/admin-settings/products-csv-format.png)
</Frame>

Here is the CSV file format for you need to follow for the **Collections**:

- Column 1: Collection ID
- Column 2: Collection Name
- Column 3: URL

<Frame>
  ![Collections CSV Format](/images/product-docs/admin-settings/collections-csv-format.png)
</Frame>

<Frame>
  ![CSV Upload Success](/images/product-docs/admin-settings/csv-upload-success.png)
</Frame>

To download or delete the previously uploaded file, click on the **Download Current File** option or the **Delete Icon**. The latest uploaded CSV file will be downloaded instantly.

<Frame>
  ![Download Delete CSV](/images/product-docs/admin-settings/download-delete-csv.png)
</Frame>
