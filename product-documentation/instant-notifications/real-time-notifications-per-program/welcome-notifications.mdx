When a new customer signs up for your loyalty program, ensure that they receive a warm welcome to get them started on the journey you have set for them.

## How to Create Real-Time Notifications for New Signups

You can access the notifications' global settings in the **Gameball Admin Dashboard**.

- From the left bar, Click **Notifications.**
- Choose between **In-App, Emails, & SMS.**
- Scroll to **Welcome,** and select **New Signup.**

<Frame>
![Welcome navigation](/images/product-docs/instant-notifications/real-time-notifications-per-program/welcome-navigation.png)
</Frame>

Start writing the message itself. Construct a heading and a body for the notification. In case you are creating an email notification, write a clear subject line and body. [Learn how to Write an email notification](/product-documentation/instant-notifications/email-notifications/sending-achievement-emails).

## Script Update for New Signup Notifications

You can add the transaction notifications script for all notifications through this page:

### New Signup

Send real-time transaction notifications for every new signup that takes place. Welcome your new customers and inform them of any details they should know at this stage.

### Script Variables for Cashback Notifications

To send more personalized, more relevant, and clearer notifications, add variables in between your messages by clicking the **+ icon**.

<Frame>
![Script variables](/images/product-docs/instant-notifications/real-time-notifications-per-program/script-variables-welcome.png)
</Frame>

You can use variables in the scripts like the ones below:

- **Customer Variables:** First Name, Last Name, Display Name, Customer's Email, Referral Link, Mobile Referral Link.
- **Variables:** Currency, Points Name.

### Preview and Change Notification Icon

Before activating the notification, you might need to take a look at what the notification will look like for customers. You can easily do so by clicking **Preview** on the top right of the edit page.

<Frame>
![Preview notification](/images/product-docs/instant-notifications/real-time-notifications-per-program/preview-notification-welcome.png)
</Frame>

When clicking on preview, you will be able to see what the notification looks like for customers. You can edit the notification icon and upload one by choosing **Upload Icon**, then click **Upload Icon**.

<Frame>
![Upload icon dialog](/images/product-docs/instant-notifications/real-time-notifications-per-program/upload-icon-dialog-welcome.png)
</Frame>

<Frame>
![Notification icon preview](/images/product-docs/instant-notifications/real-time-notifications-per-program/notification-icon-preview-welcome.png)
</Frame>

Click **Update** from the bottom right.

### Status Update for Cashback Notifications

You can enable or disable the notification through the **Active Toggle**.

<Frame>
![Active toggle](/images/product-docs/instant-notifications/real-time-notifications-per-program/active-toggle-welcome.png)
</Frame>

<Note>
Do not forget to click on "**Save**" before you exit.
</Note>
