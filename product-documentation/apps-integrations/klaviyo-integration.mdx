## Why Integrate Klaviyo with Gameball?

By integrating Klaviyo with Gameball, you unlock the ability to target and segment customers based on their loyalty to your app. This integration allows you to:

  Segment based on Gameball customer attributes such as points balance and tier.

<Card title="Send Personalized Emails" icon="envelope">

<Card title="Track and Trigger Campaigns" icon="bolt">

<Card title="Trigger Automated Flows" icon="rotate">

## How to Configure Klaviyo on Gameball

<Steps>
<Step title="Ensure Your Widget is Published">
  <Warning>
  Not publishing the widget could impact the integration process. Ensure it's live before proceeding.
  </Warning>
</Step>

<Step title="Access the Gameball Admin Dashboard">
  Navigate to **Integration** in the left menu.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-65.png" />
  </Frame>
</Step>

<Step title="Add Klaviyo Integration">
  Click on **Connect** under **Klaviyo**. If you are on Shopify, make sure Klaviyo is installed through the Shopify App Store.
</Step>

<Step title="Create an API Key in Klaviyo">
  In your Klaviyo account, go to the top right corner and click **Account Settings > Settings > API Keys**. Create a new API key and ensure full access for each API scope.

  <Warning>
  Failing to grant full access may prevent the integration from functioning properly.
  </Warning>
</Step>

<Step title="Copy and Paste the API Key into Gameball">
  Once you've generated the API key in Klaviyo, copy it. Go back to your Gameball dashboard, paste the key, and click **Next**.
</Step>

<Step title="Set Up a Webhook in Klaviyo">
  Webhooks allow Gameball to send real-time data to Klaviyo.

  1. Log in to your **Klaviyo Dashboard**.
  2. Go to **Account Settings > Webhooks**.
  3. Click **Create Webhook**:
     - Provide a name for the webhook (e.g., "Gameball Integration").
     - Enter the **Webhook URL** provided by Gameball in the integration section.
  4. Save the webhook.
</Step>

<Step title="Map Gameball Events to Klaviyo">
  Gameball tracks various customer events. Map these events to Klaviyo to use them in your campaigns.

  1. In the Gameball Dashboard, go to **Integrations**.
  2. Select **Klaviyo** from the list of available integrations.
  3. Map the following events to Klaviyo:
     - **Coupon Created**
     - **Coupon Redeemed**
     - **Reward Campaign Achieved**
     - **Order Placed**
  4. Confirm the event mappings and save.
</Step>

<Step title="Sync Customer Data">
  Syncing customer data ensures Klaviyo receives real-time updates about customer actions.

  1. In the Gameball Dashboard, navigate to **Integrations > Klaviyo**.
  2. Enable the **Customer Sync** toggle.
  3. Choose the customer attributes to sync:

  | Attribute | Attribute |
  |---|---|
  | Customer ID | Tier Progress |
  | Date of Birth | Tier ID |
  | Join Date | Completed Reward Campaigns |
  | Tier Name | Referral Code |
  | Wallet Points | Referral Link |
  | Pending Points | Tags and Segments |

  4. Save your settings.
</Step>

<Step title="Check Out Synced Customers">
  In the profile section, you will find a table with all the automatically synced Gameball customers.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-66.png" />
  </Frame>
</Step>

<Step title="Verify Custom Properties in Klaviyo">
  In Klaviyo, you can view the Gameball customer attributes under **Profile > Information > Custom Properties** of a specific customer.
</Step>

<Step title="Use Gameball Attributes in Campaigns">
  When creating a campaign in Klaviyo, you can insert Gameball attributes by selecting **Insert Property**.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-67.png" />
  </Frame>
</Step>

<Step title="Trigger Flows Based on Gameball Events">
  In addition to attributes, you can trigger automated flows in Klaviyo based on real-time Gameball events:

  - **Coupon Created**
  - **Coupon Redeemed**
  - **Reward Campaign Achieved**
  - **Order Placed**
  - **Points Redeemed**

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-68.png" />
  </Frame>

  Each event will be synced in real-time, allowing you to send highly targeted campaigns and flows in Klaviyo.
</Step>
</Steps>
