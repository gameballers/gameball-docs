Gameball and Insider integration allows you to leverage customer engagement data from Gameball to power your email marketing campaigns in Insider. Follow this guide to seamlessly connect the two platforms and maximize the impact of your loyalty and marketing efforts.

## Why Integrate Insider with Your Gameball Account?

By integrating Insider with Gameball, you unlock the ability to target and segment customers based on their loyalty to your app. This powerful integration allows you to:

<CardGroup cols={3}>
  <Card title="Segment Contact Lists" icon="filter">
    Segment contact lists based on Gameball customer attributes (e.g., points balance, tier), and events created.
  </Card>
  <Card title="Send Personalized Emails" icon="envelope">
    Include loyalty-related data, such as customer tier and reward points, directly in your email templates.
  </Card>
  <Card title="Track and Trigger Campaigns" icon="bolt">
    Track and trigger campaigns based on specific customer events, such as coupons created, orders placed, and reward achievements.
  </Card>
</CardGroup>

## Data Import Scenarios

The integration between Gameball and Insider ensures that your customer data is always up-to-date in Insider. Gameball's customer attributes will be imported into Insider whenever a customer is created or performs any action.

<AccordionGroup>
  <Accordion title="Customer Attributes Sync to Insider" icon="user">
    **Trigger:** Upon Customer Creation or after any new action by the customer (e.g., achievement, transaction, points expiry, etc.)

    Gameball's customer attributes will be imported and updated once a customer is created or performs any action.

    **Imported Data:**
    - ID
    - Email
    - Join Date
    - Mobile Number
    - Birthday
    - Gender
    - Points balance
    - Pending points
    - Current level name
    - Current level order
    - Next Level
    - Next level threshold
    - Level Progress
    - Progress to achieve next level
    - Last order awarded points
    - Referral Code
    - Referral Link
    - Tags
    - Segments
    - RFM segments
    - Latest created coupon
    - Latest used coupon
    - Count of rewarded campaigns
    - Starting Store (store of account creation)
    - Connecting Store (store of the last transaction)
    - Store of First Transaction
    - Average Basket
    - Number of Transactions
    - Transactions Amount in the Last Rolling Year
    - Total Net Amount of Transactions
    - Unit per Transaction (Unit Per Trx)
    - Earned Points
    - Redeemed Points
    - Expired Points
    - Date of First Transaction
    - Date of Last Transaction
  </Accordion>

  <Accordion title="Update & Notify Insider with Every Order Placed" icon="bag-shopping">
    **Trigger:** "Order Placed" Event — Order Placed by Customer & Received by Gameball

    Once an order is placed by a customer, the following order attribute data is sent to Insider:
    - Order ID
    - Order Date
    - Total Paid
    - Total Price
    - TotalShipping
    - TotalTax
    - TotalDiscount
    - Discount Codes
    - Extra
    - Redeemed Amount
    - Rewarded Points
    - Equivalent Points Value
  </Accordion>

  <Accordion title="Update & Notify Insider with Every Coupon Created" icon="ticket">
    **Trigger:** "Coupon Created" Event — Coupon Created for the Customer

    Once a coupon is created for the customer, the following Coupon details will be available on Insider linked to the customer's data:
    - Coupon Code
    - Type
    - Value
    - Coupon Name
  </Accordion>

  <Accordion title="Update & Notify Insider with Every Coupon Burnt" icon="fire">
    **Trigger:** "Coupon Used" Event — Customer burnt the coupon while placing an order.

    Once a coupon is used by the customer, the following details will be available on Insider:
    - Coupon Code
    - Type
    - Value
    - Order ID
    - Coupon Name
    - Channel (POS, Online, etc.)
    - Order Quantity
  </Accordion>

  <Accordion title="Update & Notify Insider with Every Reward Campaign Achieved" icon="trophy">
    **Trigger:** "Reward Campaign Achieved" Event — Customer Achieved a Reward Campaign

    Once a Reward Campaign is achieved by the customer, the following details will be available on Insider:
    - Internal Name
    - Name
    - Reward (Points, Points&Score, Coupon Values)
    - Coupon Type (in case the reward was a coupon)
    - Equivalent Value of Rewarded points (in case the reward was in the form of points, Points&Score)
    - Achievement Date
  </Accordion>

  <Accordion title="Notify Insider upon Points Redemption" icon="coins">
    **Trigger:** "Points Redeemed" Event — Customer redeemed their saved points.

    Each `points_redeemed` event includes:
    - Redeemed points
    - Redemption amount
    - Redemption transaction ID (if available)
  </Accordion>

  <Accordion title="Update & Notify Insider with Customer Deletion" icon="user-minus">
    **Trigger:** Customer was removed from Gameball

    Once a customer is deleted from Gameball, after **24 hours**, the customer gets deleted from Insider.

    <Note>
      You have the option to disable the deletion feature from Insider.
    </Note>
  </Accordion>

  <Accordion title="Notify Insider for Each Line Item Bought" icon="list">
    **Trigger:** Order received by Gameball

    Send event call to Insider "Purchase" with all attributes of the Item and the order ID:
    - orderId
    - Product Title
    - Unit Price
    - Vendor Name
    - Product ID
    - Weight
    - Quantity
    - SKU
    - Collections array
    - Tags array
    - Taxonomy array
    - "Extra." Any added attributes to this object will be sent inside the event
  </Accordion>
</AccordionGroup>

## How to Configure Insider on Gameball

Follow these steps to connect Insider with Gameball and make the most of your loyalty data:

<Steps>
  <Step title="Ensure Your Widget is Published">
    Make sure your Gameball widget is published and live before proceeding. Not publishing the widget could impact the integration process.
  </Step>

  <Step title="Access the Gameball Admin Dashboard">
    Navigate to **Integrations** in the menu. Scroll down to the Insider app and click on **Connect**.

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-47.png" />
    </Frame>
  </Step>

  <Step title="Copy and Paste Partner Name into Gameball">
    In your Insider account, go to the top right corner, click **Settings > "App Name" Settings > Account Settings**. Copy the **Partner Name**.

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-48.png" />
    </Frame>

    Go back to your Gameball dashboard and paste the Name.

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-49.png" />
    </Frame>
  </Step>

  <Step title="Create an API Key in Insider">
    In your Insider account, go to **Settings > "App Name" Settings > Integration Settings**.

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-50.png" />
    </Frame>

    Scroll down and click **Generate API key**.

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-51.png" />
    </Frame>

    Choose **Unified Customer Database** and click **Next**.

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-52.png" />
    </Frame>

    <Warning>
      Choosing a different type for the API Key may prevent the integration from functioning properly.
    </Warning>
  </Step>

  <Step title="Copy and Paste the API Key into Gameball">
    Once you've generated the API key in Insider, copy it. Go back to your Gameball dashboard, paste the key, and click **Next**.

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-53.png" />
    </Frame>
  </Step>

  <Step title="Map Gameball Events to Insider">
    Gameball tracks various customer events (e.g., points earned, redeemed, tiers achieved). Map these events to Insider to use them in your campaigns.

    1. In the Gameball Dashboard, go to **Integrations**.
    2. Select **Insider** from the list of available integrations.
    3. Map the following events to Insider:
       - **Coupon Created**
       - **Coupon Redeemed**
       - **Reward Campaign Achieved**
       - **Order Placed**
    4. Confirm the event mappings and save.
  </Step>

  <Step title="Sync Customer Data">
    Syncing customer data ensures Insider receives real-time updates about customer actions.

    1. In the Gameball Dashboard, navigate to **Integrations > Insider**.
    2. Enable the **Customer Sync** toggle.
    3. Choose the customer attributes to sync:
       - **Customer ID**
       - **Date of Birth**
       - **Join Date**
       - **Tier Name**
       - **Wallet Points**
       - **Pending Points**
       - **Tier Progress**
       - **Tier ID**
       - **Completed Reward Campaigns**
       - **Referral Code**
       - **Referral Link**
       - **Tags and Segments**
    4. Save your settings.
  </Step>

  <Step title="Check Out Synced Customers">
    In the **User Profiles** section, you will find a table with all the automatically synced Gameball customers.

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-54.png" />
    </Frame>
  </Step>

  <Step title="Trigger Flows Based on Gameball Events">
    In addition to attributes, you will see real-time Gameball events such as:
    - **Coupon Created**
    - **Coupon Used**
    - **Reward Campaign Achieved**
    - **Order Placed**
    - **Purchase**

    <Frame>
      <img src="/images/product-docs/apps-integrations/apps-integrations-55.png" />
    </Frame>

    Each event will be synced in real-time, allowing you to send highly targeted campaigns and flows in Insider.
  </Step>
</Steps>
