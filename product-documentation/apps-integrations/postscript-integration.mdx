## Integration Highlights

This integration enables you to:

1. **Add Loyalty Data to Subscriber Profiles:** Gameball loyalty data, like points balance and loyalty tier, appears as custom properties within Postscript.
2. **Automate Messaging with Loyalty Triggers:** Trigger flows based on loyalty events, such as points earned.
3. **Segment for Targeted Campaigns:** Use Gameball properties to create specific customer segments in Postscript, ensuring campaigns reach the right audience.
4. **Personalize Communication Campaigns:** Customize SMS content using Gameball properties to deliver more personalized, engaging messages.

## Setup Guide

<Steps>
<Step title="Prerequisites">
  Before starting the integration, ensure the following:

  - You have an active **Gameball** account.
  - You have an active **Postscript** account.
  - Access to admin permissions in both platforms.
</Step>

<Step title="Copy the API Key from Postscript">
  Go to the Postscript dashboard and follow these steps:

  1. Log in to your **Postscript Dashboard**.
  2. Go to **Settings > API**.
  3. Click **Create Security Key Pair**.
  4. Once the new key is generated, enter the label.
  5. Click **Show** and copy the private key.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-33.png" />
  </Frame>
</Step>

<Step title="Install Postscript on Gameball">
  1. Log in to your **Gameball Dashboard**.
  2. Navigate to **Integrations** from the left menu.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-34.png" />
  </Frame>

  3. Select **PostScript** and click **Connect**.
  4. Paste the **Generated API Key** from PostScript and click **Next**.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-35.png" />
  </Frame>
</Step>
</Steps>

## Synced Events and Data

<Tabs>
<Tab title="Events Synced to Postscript">
  Gameball tracks various customer events and automatically maps these to Postscript for use in your campaigns:

  - **Coupon Created**
  - **Coupon Redeemed**
  - **Reward Campaign Achieved**
  - **Order Placed**
</Tab>

<Tab title="Synced Customer Attributes">
  These customer attributes are automatically synced to Postscript in real time:

  | Attribute | Property Key |
  |---|---|
  | Customer ID | `gb_id` |
  | Date of Birth | `gb_birthday` |
  | Join Date | `gb_join_date` |
  | Tier Name | `gb_level_name` |
  | Wallet Points | `gb_wallet_points` |
  | Pending Points | `gb_pending_points` |
  | Tier Progress | `gb_level_name` |
  | Tier ID | `gb_level_id` |
  | Completed Reward Campaigns | `gb_completed_challenges` |
  | Referral Code | `gb_referral_code` |
  | Referral Link | `gb_referral_link` |
  | Tags and Segments | `gb_tags_and_segments` |
</Tab>
</Tabs>

## Creating Segments Based on Loyalty Data

Segmentation allows you to send targeted messages to a specific group of subscribers based on the existing Gameball attributes.

<Steps>
<Step title="Open Segments in Postscript">
  From your Postscript Dashboard, go to **Messaging > Segments > Create Segment**. Enter the name of your new segment.
</Step>

<Step title="Set Segment Criteria">
  Below **Segment Criteria**, click on the pre-selected filter to replace it.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-36.png" />
  </Frame>
</Step>

<Step title="Select a Gameball Attribute">
  From the dropdown menu, select any of the Gameball available attributes.

  - Optionally, click **+ Add Rule** to add an `or` connector within the same filter.
  - Optionally, add more filters for your segment if needed.
</Step>

<Step title="Save the Segment">
  Click **Save Segment**.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-37.png" />
  </Frame>
</Step>
</Steps>
