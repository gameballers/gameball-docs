## Plan Availability

| Platform | Plan |
|---|---|
| Shopify | Starter, Pro, and Guru |

With Gameball's integration into Shopify Flow, you can automate and enhance your customer loyalty and engagement programs more effectively. This article walks you through the available triggers and actions, how to set them up within Shopify Flow, and practical use case examples.

---

## Available Triggers

| Trigger Name | Description |
|---|---|
| **Customer Tier changed (Up/Down)** | Activates when a customer tiers up or down, either manually or automatically. _e.g., When a customer's tier changes to the Gold tier._ |
| **Reward Campaign Achieved** | Activates when a customer achieves a new Reward Campaign, either manually or automatically. _e.g., When a customer earns a new reward campaign when they complete their First Order._ |
| **Customer Entered on a New Segment** | Activates once a customer is added to a new Segment. _e.g., When a customer is placed in the Inactive segment._ |
| **Customer Added a Birthday** | Activates once a customer adds their birthday to Gameball's widget. |

## Available Actions

| Action Name | Description |
|---|---|
| **Send an Event** | Sends a custom event to Gameball. _e.g., A customer completed a survey, so we send this event (Survey_Completed) to Gameball._ **Note:** Only available on the Guru Plan. |
| **Reward a Campaign** | Rewards a customer with a specified Reward Campaign manually. _e.g., Rewarding a customer with the 'Completed First Order' Reward Campaign._ |
| **Deduct Points** | Deducts a specified number of points from a customer's account for a given reason. _e.g., Deducting 50 points from a customer's balance for a reward they got in-store._ |
| **Adjust Customer Tier** | Adjusts a customer's tier to a specified tier name. _e.g., Adjusting the customer tier from the Bronze tier to the Gold tier._ |
| **Add Points** | Adds a specified number of points to a customer's account for a given reason. _e.g., Adding 100 points for a customer for adding a review on your product._ |

## Actions Setup Details

<AccordionGroup>
<Accordion title="Send an Event" icon="bolt">
  This action allows you to send a custom event to Gameball. Fill in the following attributes and add a description for documentation purposes.

  **Attributes:**

  - **Customer Email**: Skip the email if the trigger already passes the ID. Select it from "Add a variable".

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-106.png" />
  </Frame>

  - **Event Name**: The event name to be sent to Gameball.
  - **Event Attribute Name X**: The event attribute name for the event. You can add up to 5 attributes.
  - **Event Attribute Value X**: The event attribute value for the event.

  <Note>
  This action is only available on the Guru Plan.
  </Note>
</Accordion>

<Accordion title="Reward a Campaign" icon="trophy">
  This action rewards a customer with a specified Reward Campaign manually.

  **Attributes:**

  - **Customer Email**: Skip the email if the trigger already passes the ID. Select it from "Add a variable".

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-107.png" />
  </Frame>

  - **Campaign ID**: Copy the ID next to the title on the campaign details page in Gameball.

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-108.png" />
  </Frame>
</Accordion>

<Accordion title="Deduct Points" icon="minus">
  This action deducts a specified number of points from a customer's account for a given reason.

  **Attributes:**

  - **Customer Email**: Skip the email if the trigger already passes the ID. Select it from "Add a variable".

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-109.png" />
  </Frame>

  - **Points to be deducted:** Number of points to deduct from the customer.
  - **Reason**: Reason for deducting the points.
</Accordion>

<Accordion title="Adjust Customer's Tier" icon="arrow-up-down">
  This action adjusts a customer's level to a specified level name. Choose the level behavior and provide a reason for the adjustment.

  **Attributes:**

  - **Customer Email**: Skip the email if the trigger already passes the ID. Select it from "Add a variable".

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-110.png" />
  </Frame>

  - **Tier Order:** The tier number in Gameball (e.g., 1, 2, 3, etc.).
  - **Tier Behavior:** How long a customer remains in the new tier:

  | Option | Description |
  |---|---|
  | **System Evaluate** | Gameball evaluates the customer's tier according to the chosen tiering-up method and expiry duration |
  | **Forever** | The customer stays on this tier unless manually assigned to a new tier |
  | **Temporarily till Specific Date** | The customer stays on the mentioned tier until a specific date |

  - **Temporarily Specific Date:** If you selected `temporarily_till_specific_date`, add the date in this format: `yyyy-MM-dd`.
  - **Reason**: The reason for the tier adjustment.
</Accordion>

<Accordion title="Add Points" icon="plus">
  This action adds a specified number of points to a customer's account for a given reason.

  **Attributes:**

  - **Customer Email**: Skip the email if the trigger already passes the ID. Select it from "Add a variable".

  <Frame>
    <img src="/images/product-docs/apps-integrations/apps-integrations-111.png" />
  </Frame>

  - **Points to be added:** Number of points to add to the customer.
  - **Reason**: Reason for adding the points.
</Accordion>
</AccordionGroup>

## Setting Up Gameball Triggers and Actions in Shopify Flow

<Steps>
<Step title="Access Shopify Flow">
  From your Shopify admin, go to **Apps > Shopify Flow**.
</Step>

<Step title="Create a Workflow">
  Click on **Create Workflow**, then choose a trigger from the Gameball triggers listed above.
</Step>

<Step title="Configure the Trigger">
  Set up the trigger by selecting the appropriate variables and conditions.
</Step>

<Step title="Add an Action">
  Select an action from the Gameball actions listed above and configure the action with the necessary details.
</Step>

<Step title="Test and Activate">
  Test your workflow to ensure it works as expected. Once confirmed, activate the workflow.
</Step>
</Steps>

## Use Cases

<AccordionGroup>
<Accordion title="Rewarding a Campaign for Customers Who Gave a Review" icon="star">
  You can thank customers who take the time to leave reviews on your products. With Gameball and Shopify Flow, you can automate this process:

  - **Trigger:** Set up an external rewards collection app like Reviews.io to trigger the action whenever a customer leaves a review.
  - **Action:** Use Gameball's **Reward a Campaign** action to ensure customers feel appreciated for their feedback.

  Here is a live example on how to reward customers with a reward campaign when they leave a positive review on Reviews.io. [Check it out.](/product-documentation/apps-integrations/shopify-flow/reviews-io-integration)
</Accordion>

<Accordion title="Adding Points to Customers Who Filled Out a Survey" icon="clipboard-list">
  Encourage customers to provide valuable insights by filling out a survey:

  - **Trigger:** Use an external survey app such as Okendo to trigger the action whenever a customer completes a survey.
  - **Action:** Gameball will automatically add points to the customer's account when you use the **Add Points** action, incentivizing them to participate in more surveys.

  Here is a live example on how to add points to customers when they fill a survey on Okendo. [Check it out.](http://help.gameball.co/en/articles/9750783-okendo-app-integration-through-shopify-flow)
</Accordion>

<Accordion title="Celebrate a Customer's 3-Year Anniversary with Your Store" icon="cake-candles">
  Recognizing milestones can significantly enhance customer loyalty:

  - **Trigger:** Set a date trigger to automatically recognize when a customer has been with your store for three years.
  - **Action:** Use Gameball's **Send Event** action and use the event to trigger a personalized communication campaign — including a congratulatory message and a special reward.

  This personalized approach makes customers feel valued and strengthens their connection with your brand.
</Accordion>
</AccordionGroup>
