## Intro

Motivate customers to increase their order value by offering rewards for larger purchases. Each time the customer exceeds their highest order value, they will earn a reward that you set up. This tactic can help boost average order value and maximize revenue per transaction.

<Frame>
  ![Exceed highest order value campaign widget](/images/product-docs/rewards-campaigns/templates/purchase-order-incentives/exceed-highest-order-widget.png)
</Frame>

## Creation Experience

**How to Set Up**

1. Open Gameball
2. Click **Add Reward Campaign**
3. Select **Exceed your highest order value** template
4. Edit the campaign's setup
5. Click **Set Live**

**Trigger**

1. This campaign rewards customers for placing orders.
2. Each time a customer places an order that exceeds their previous highest order value, they earn a reward.
3. You can set a minimum order value to activate the campaign (e.g., the order must exceed a specific amount to qualify, such as $100).

<Frame>
  ![Exceed highest order value trigger setup](/images/product-docs/rewards-campaigns/templates/purchase-order-incentives/exceed-highest-order-trigger.png)
</Frame>

**Repeatability**

- Customers can earn this reward unlimited times, as long as each new order exceeds their previous highest order value.
- This setting cannot be customized.

**More Setup**

This template uses the standard reward campaign setup. See the [Generic Setup Guide](/product-documentation/rewards-campaigns/start-from-scratch/setup-reward-campaign) for details.

## End User Experience

**How it works**

1. Customer places an order with an order value higher than the customer's previous highest purchase.
2. Customer gets rewarded immediately (points, discount, badge -- based on your setup).
3. The customer is notified via the widget or email, depending on your configuration. [Learn more](/product-documentation/rewards-campaigns/start-from-scratch/setup-reward-campaign)

## Important Notes

<Note>
**Earning Rules**

1. This campaign only triggers on placed order events.
2. Customers can earn the reward unlimited times, as long as each new order exceeds their previous highest value.
</Note>
