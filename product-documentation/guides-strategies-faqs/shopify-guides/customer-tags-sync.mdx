## How Tag Syncing Works Between Gameball and Shopify

If you are a Shopify user using Gameball, all customer tags on Shopify will be synced with Gameball automatically.

<Frame>
  <img src="/images/product-docs/guides-strategies-faqs/shopify-tags-sync.png" alt="Shopify Tags Sync" />
</Frame>

<Tabs>
<Tab title="Auto-Tagging">
The process works as follows:

1. Add a tag to your customer on Shopify.
2. The tag will be automatically created on Gameball (if it is a new tag).
3. This tag will be added to the customer on Gameball.
</Tab>

<Tab title="Auto-Untagging">
The process works as follows:

1. Remove a tag from your customer on Shopify.
2. This tag will be removed automatically from the customer on Gameball.
</Tab>
</Tabs>

<Note>Sometimes, the auto-tagging and auto-untagging take time, between 1–5 minutes.</Note>
